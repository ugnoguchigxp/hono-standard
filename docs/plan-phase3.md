# Phase 3: Web 閲覧強化 実装計画

## 概要

React Web の閲覧機能を強化するフェーズ。
週次・月次グラフ、日次サマリの改善、履歴画面の編集機能を実装する。

## 前提条件

- Phase 1, 2 が完了していること
- 健康記録 API が稼働していること
- Flutter から記録データが投入されていること

## 期間

4 週間（目安）

## 成果物

- 週次・月次推移グラフ
- 改善された日次サマリ画面
- 編集可能な履歴画面
- Flutter と同じ集計値の表示確認

## リスクと対策

| リスク | 影響 | 対策 |
|--------|------|------|
| グラフライブラリの学習コスト | 開発遅延 | 事前にサンプル実装で検証 |
| 大量データでのパフォーマンス低下 | UX 悪化 | データの集約、ページネーション |
| Flutter との集計ロジック差異 | ユーザー混乱 | バックエンドで集計を一元化 |

---

## 依存関係

```
Week 1 (グラフ基盤)

Week 2 (推移グラフ)
    ↓
```

Week 3 の履歴編集（バックエンド）は Week 1 と並行可能。
