## [0.1.0] - 2026-01-06

### Added
- Initial release of `@gxp/design-system`.
- Setup Vite library mode build configuration.
- Setup Storybook for component development.
- Setup Vitest for unit testing.
- Setup Biome for linting and formatting.
- Implemented 47+ UI components (Button, Input, Card, Modal, etc.).
- Implemented 11 built-in themes.
- CSS variables support for styling.
- Tailwind CSS v4 support via preset.
